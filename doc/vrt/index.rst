libiio VITA49.2 Backend Documentation
=====================================

Welcome to the documentation for the `libiio` VITA 49.2 Backend integration.

This module provides a robust, generic standard backend capable of communicating 
with remote IF context devices that encapsulate I/Q data directly in VITA 49.2 
VRT (VITA Radio Transport) packets.
